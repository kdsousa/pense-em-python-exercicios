# Pense em Python 🐍

Este repositório contém minhas resoluções de exercícios e anotações de estudo baseadas no livro **"Pense em Python: Pense Como um Cientista da Computação"** de Allen B. Downey.

## 🎯 Objetivo
O objetivo deste projeto é praticar a lógica de programação em Python, compreender estruturas de dados fundamentais e registrar minha evolução na jornada de aprendizado.

## 🗂️ Estrutura do Repositório
Cada capítulo do livro possui sua própria pasta contendo os exercícios propostos:
- `capitulo-01/`: A jornada do programa.
- `capitulo-02/`: Variáveis, expressões e instruções.
